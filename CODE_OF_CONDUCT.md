# Code of Conduct

This project follows a simple expectation: be respectful, factual, and constructive.

## Expected Behavior

- Use welcoming and professional language.
- Focus criticism on code, documentation, and reproducibility.
- Respect that this is a hackathon project with time constraints.
- Keep discussions evidence-based, especially when evaluating ranking changes.

## Unacceptable Behavior

- Harassment, insults, or personal attacks.
- Sharing private data or credentials.
- Misrepresenting generated results as manually produced.
- Pressuring contributors to hide reproducibility or validation problems.

## Enforcement

Maintainers may remove comments, close issues, or block participation when behavior is harmful to the project or its contributors.
